Programa de Optimizador de PDFs en base a los requerimientos de los procesos aduaneros en México.

Este programa toma todas las imagenes de un PDF y las pasa a 300pp, cambia el color a escala de grises y modifica el peso para que no supere los 3MB.

Si es que el PDF contiene muchas imagenes, es posible que en algunos pdf algunas imagenes se muevan a la parte de atras, por lo que con un software como adobe acrobat, las regreso a su posición real.
